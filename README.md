# LearnFi
![afa3b254edc6e775b1834e04f769def3](https://github.com/user-attachments/assets/e2f005a3-220c-48b1-b0db-ad5f42e8a2e6)

LearnFi，一款基于 Web3 技术打造的「学习即价值」平台。我们打破了传统学习工具 “仅记录、无回报” 的局限，通过区块链确权、代币激励、NFT 成就体系，让每一分钟学习都能转化为可积累、可流转的数字资产，同时搭建沉浸式学习社区，让学习更有动力、更有价值。
## ✨核心定位
本项目面向初高中学生、大学生、考研 / 考公 / 考证等备考人群，以及所有希望通过学习提升自我的用户，解决 “学习自制力差、缺乏持续动力、学习成果无价值沉淀” 三大痛点。

不同于传统学习 App 仅提供计时、打卡功能，LearnFi 将用户的学习行为通过智能合约确权，转化为代币积分和 NFT 资产，让 “努力学习” 不仅能收获知识，更能获得实实在在的数字价值回报；同时通过链上社区互动，营造 “互助监督、价值共享” 的学习氛围。

## 🧩使用流程
LearnFi 操作极简，无需复杂设置，3 步即可快速上手：
1. 登录注册
2. 质押积分（代币）后开启学习计时
3. 完成计时
4. 领取NFT
5. 个人中心查看学习记录，成就墙（NFT勋章）

## 🏅未来规划：持续构建学习价值生态
1. 学习：上线多人频道自习室，可同时容纳多人同时在线学习；增加积分（代币）鼓励机制；显示用户状态（学习中/休息中），增加番茄钟和小习惯计时，增加深度学习功能；
2. 社交：增加用户之间的聊天功能，实现快速交流；可发布学习动态记录心得，也可浏览社区其他用户的动态，点赞、评论、打赏。
